# Editor

A simple text editor

## Installation

1. Get Java
Check if Java is installed
```
$ java -version
```
If it isn't, download [Java](https://java.com/download)

2. Clone the repository
```
$ git clone https://github.com/isirk/Editor
```

3. Start Editor
```
$ java src/Main.java
```
